Este repositorio contiene el código de Lean utilizado para el TFG, Demostración Automática en Lean: Álgebras de Procesos, del Grado de Matemáticas de Pablo Saurina Martínez (2022).

El archivo con el código es ProcessAlgebra.lean y se encuentra en la carpeta src.

La versión de Lean utilizada es 3.35.1
